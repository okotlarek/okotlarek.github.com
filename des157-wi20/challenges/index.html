Why we use the Normalize CSS Script

Instead of resetting default browser settings, the Normalize CSS Script accounts for differences in browser defaults and normalizes them for efficiency and consistency. It also corrects bugs and browser inconsistencies.

What is Semantic Markup

Semantic markup refers to writing HTML in a way that makes sense to both computers and humans. Computers need languages with standardized keywords to function. By using words that have semantic meaning to humans, it is more instuitive to learn and understand.
